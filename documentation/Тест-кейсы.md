## TC-001
| Поле | Значение |
|:-----|:---------|
| **Название** | Запрос метода без параметров, получение в теле ответа результаты для MOW (параметр origin по-умолчанию) |

| **Шаги** | 1. В папке "/latest_prices" найти запрос "Empty request".<br>2. Выполнить запрос. |
| **Ожидаемый результат** | 200 OK. В теле ответа получены данные по ценам на билеты с вылетом из Москвы (при отсутствии указанных параметров вылета и прилета - origin и destination).|
